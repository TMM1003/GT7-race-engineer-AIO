## Model Selection and Rationale

This study evaluates supervised regression models for corner-level performance attribution using structured telemetry-derived features. Model selection is guided not only by predictive accuracy, but also by interpretability, stability, and the ability to compare learned behavior against deterministic motorsport heuristics. Rather than surveying a broad set of unrelated architectures, the work focuses on three complementary model classes that expose meaningful differences in learning behavior under identical telemetry representations.

### Ridge regression
is included as the primary linear baseline. Its role is to determine whether corner-level time loss can be explained sufficiently by linear combinations of engineered telemetry features. Because Ridge applies L2 regularization, it provides a stable and interpretable parametric model while reducing sensitivity to collinearity among related telemetry variables. This makes it well suited for evaluating whether simple, domain-grounded feature relationships already account for much of the observed performance variation.

### RandomForestRegressor
is used as the first nonlinear baseline. It allows the study to test whether conditional feature interactions and threshold-like behavior improve explanatory capacity beyond what is possible with a linear model. In the telemetry setting, this is especially relevant because braking, throttle application, speed retention, and curvature effects may interact in ways that are not well represented by a single linear surface. Random Forest also remains relatively accessible from an interpretability perspective through permutation importance and fold-level comparison.

### CatBoostRegressor 
is included as the strongest practical tabular model in the comparison. The corner-level dataset in this project is structured as a supervised tabular regression problem with engineered features, moderate dataset size, and some missing values. Gradient-boosted decision tree methods are well suited to this setting, and CatBoost is selected because it is robust on heterogeneous tabular data, captures nonlinear dependencies effectively, and handles missing values cleanly with minimal preprocessing burden. Its inclusion allows the study to test whether a modern boosting-based learner materially improves predictive performance or explanatory consistency relative to both Ridge and Random Forest.

Together, these three models provide a deliberate progression in representational capacity. Ridge evaluates linear sufficiency, Random Forest evaluates classical nonlinear interaction learning, and CatBoost evaluates a stronger modern tabular learner under the same data conditions. This comparative structure is aligned with the research objective of understanding how model class influences behavior, interpretability, and generalization in corner-level motorsport telemetry analysis.

Model evaluation is performed under grouped cross-validation to reduce leakage across related samples from the same run structure. Performance is assessed using RMSE and R², while interpretability is examined through model coefficients, permutation importance, and feature-attribution analysis where appropriate. In addition, feature-set ablation is used to compare heuristic-only telemetry metrics against broader engineered representations, allowing the study to determine whether learned models meaningfully augment deterministic domain knowledge.

This modeling strategy supports a central thesis objective: not merely identifying the most accurate predictor, but characterizing when simple interpretable models are sufficient, when nonlinear models add value, and whether improved predictive performance corresponds to meaningful explanatory insight rather than opaque complexity.
